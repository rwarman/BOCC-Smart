To update the firmware from a Linux system:


Ensure you’re connected direct from the computer to the controller using a single USB cable of 16’ or less.

 

Go to: home folder > file system > opt/SMART Technologies/SMART Product Drivers/bin

 

Double click on  SMARTfirmwareupdater and choose Run.
